<p>Aspose.Slides untuk .NET adalah API Pemrosesan Presentasi untuk format PowerPoint dan OpenOffice. Aspose.Slides memungkinkan aplikasi untuk membaca, menulis, melindungi, memodifikasi, dan mengonversi presentasi dalam .NET C#. Kelola teks presentasi, bentuk, bagan, tabel & animasi, tambahkan audio & video ke slide, pratinjau slide.</p>

<p>Aspose.Slides untuk .NET mendukung semua format presentasi populer. Gunakan C# untuk menggabungkan, membagi, mengkloning, dan menggunakan kembali beberapa presentasi PowerPoint, template, dan slide dengan berbagai cara. Selain itu, API menawarkan sejumlah fitur canggih seperti mencetak pada printer fisik dan menampilkan presentasi ke format tata letak tetap, HTML & gambar.</p>

<p>Aspose.Slides untuk .NET adalah API mandiri dan tidak memerlukan Microsoft PowerPoint atau perangkat lunak atau pustaka tambahan lainnya.</p>

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);
